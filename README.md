# Bento-grid-layout

BentoGrid Collection

A collection of modern Bento Grid–style landing page projects built using HTML & CSS, focused on fashion, editorial layouts, and visual storytelling.

This repository contains two independent Bento Grid projects, each with its own assets and styles.
